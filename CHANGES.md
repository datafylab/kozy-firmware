# Изменения в проекте

## Основные изменения

1. **Изменена левая панель модулей**:
   - Панель теперь изначально пуста с заголовком "MODULES"
   - Добавлена круглая кнопка с плюсиком по центру для добавления модулей
   - При нажатии на кнопку появляется меню с возможностью добавления RealSense Camera, Servo Drives и RPi Pico

2. **Добавлена функциональность управления модулями**:
   - Каждый добавленный модуль имеет кнопки "Disable/Enable" и "Remove"
   - При отключении модуль остается на панели, но становится неактивным
   - При удалении модуль исчезает с панели

3. **Добавлена функция смены темы в настройках**:
   - Созданы темы: Raw Cyber (кибер-тема), Dark (темная тема) и Light (светлая тема)
   - Добавлено подменю "Theme" в меню "Program" с выбором темы
   - Добавлен диалог настроек с выпадающим списком для выбора темы
   - Реализована функция change_theme() для переключения между темами

4. **Исправлена ошибка импорта**:
   - Исправлена ошибка импорта QAction из PySide6.QtWidgets (теперь импортируется из PySide6.QtGui)

## Файлы, которые были изменены:

- `/workspace/gui/main_window.py` - основной файл с изменениями интерфейса
- `/workspace/gui/styles.py` - добавлены новые темы
- `/workspace/gui/dialogs.py` - добавлен диалог настроек
- `/workspace/CHANGES.md` - этот файл с описанием изменений
